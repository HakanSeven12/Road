# FreeCAD Road Workbench

## Features

* Import Survey Data Files
* Export Points to File
* Terrain
* Alignment
* Profile
* Regions
* Sections
* Earthworks Calculation
* GeoLine
* Import LandXML files

## Installation

Download and extract the ZIP file into the `.FreeCAD/Mod` folder.
